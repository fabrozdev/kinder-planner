### Kinder Planner

This application is a digital version of a capacity-aware monthly planning spreadsheet for two care locations. It must support automatic alphabetical sorting, capacity validation, day-based attendance lists, monthly planning, and yearly reporting.

### Tech Stack

#### Frontend

- Angular v20
- PrimeNG
- NgRx

#### Backend

- Kotlin v2.2.21
- Spring Boot v3.5.7
- PostgreSQL
- MapStruct
